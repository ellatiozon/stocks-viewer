What is this website?

The Stock Price Viewer is a simple web application that allows users to check the latest stock prices of companies using real-time financial data.

How does it work?

This website fetches data from the Financial Modeling Prep API, which provides real-time stock market information including company names, current prices, and percentage changes.

Features

  • Search any stock by its ticker symbol (e.g. AAPL, TSLA, GOOGL)
  • Displays current price and change percentage
  • Uses an API to fetch live market data
  • Fully responsive and hosted online

About the Developer

This project was created as a small web development exercise using HTML, CSS, and JavaScript. It demonstrates API integration and form submission functionality.
